# Многопоточное программирование
Цель работы. Реализовать многопоточную свёртку изображения (количество потоков можно выбирать). С помощью операции свёртки часто реализуются различные фильтры – например, размытие или повышение резкости. После выполнения операции нужно выводить затраченное на непосредственно многопоточную свёртку время.
